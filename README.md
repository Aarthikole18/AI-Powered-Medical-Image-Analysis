# 🧠 AI-Powered Medical Image Analysis System

## 📌 Overview

This project is an AI-based system that analyzes medical images (Chest X-rays) to detect diseases like:

* COVID-19
* Pneumonia
* Normal cases

It uses Deep Learning (CNN) to classify images and assist in medical diagnosis.

---

## 🎯 Objective

To build a system that can:

* Analyze X-ray images
* Detect diseases automatically
* Assist doctors in faster diagnosis

---

## 🏥 Industry Relevance

AI-based medical imaging is used in:

* Hospitals
* Radiology centers
* Health-tech companies

This project simulates real-world AI healthcare systems.

---

## ⚙️ Tech Stack

* Python
* TensorFlow / Keras

---

## 📂 Dataset

Used Chest X-ray dataset with 3 classes:

* COVID
* Pneumonia
* Normal

---

## 🧠 Model

* Convolutional Neural Network (CNN)
* Multi-class classification (3 classes)
* Softmax output layer
